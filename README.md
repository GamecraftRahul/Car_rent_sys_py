# Car Rental Management System

A full-featured car rental management system built with Python, Tkinter, and MySQL.

## Features

*   **Car Management**: Add, update, delete, and search cars
*   **Dashboard**: Real-time statistics (Total Cars, Available, Rented, Returning Today)
*   **Search Functionality**: Filter cars by name/brand
*   **Database Integration**: MySQL backend for data storage
*   **User-Friendly GUI**: Built with CustomTkinter for modern look

## Technologies Used

*   Frontend: Python with [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)
*   Backend: MySQL Database
*   Database: `car_rental_system` database
*   Other: Tkinter, ttk, messagebox
